mee.cla.BrowserBodyView Class

Defined in: public/javascripts/views/browser-body.js:4

This class will be used to support the main view of the object browser. From here, the user will be able to browse notes, tags and tasks, using filters and sorting out results. It controls the creation of several subviews, like mee.cla.BrowserBodyTagView, mee.cla.BrowserBodyFilterView,...

Events

browser:[collection-name]:[action]

Defined in public/javascripts/views/browser-body.js:221

Event triggered on mee.dispatcher after the user clicks on an action button (flagged in the DOM with the .actions-contextual-trigger button classes). It should be listened by subviews (for example instances of mee.cla.BrowserBodyTagView) to let them relay the action to their respective model.

browser:filters:[filter-name]:remove-active

Defined in public/javascripts/views/browser-body.js:368

Event triggered on mee.dispatcher after the user decides to delete the saved tag that is currently active. This event is listened by the views of the class mee.cla.BrowserBodyFilterView. The one view related to a model with the right filter name will delete its model.
