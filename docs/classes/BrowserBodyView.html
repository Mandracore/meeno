BrowserBodyView Class

Extends Backbone.View
Defined in: public/javascripts/views/browser-body.js:14

This class will be used to support the main view of the object browser. From here, the user will be able to browse notes, tags and tasks, using filters and sorting out results. It controls the creation of several subviews, like BrowserBodyTagView, BrowserBodyFilterView,...
