mee.cla.BrowserBodyFilterView Class

Extends Backbone.View
Defined in: public/javascripts/views/browser-body-filter.js:4

This class holds the view of the filter models that are used to filter the browser's search results. This view can accept models of classes mee.cla.NoteFilter, mee.cla.TaskFilter and mee.cla.TagFilter.
